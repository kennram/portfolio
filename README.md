# Kenn Davis | Socio-Technical Architect Portfolio

## Mission
To architect and design human-centered, socio-technical systems that bridge the gap between technical complexity and human need. This portfolio serves as both a digital storefront and a live research ledger.

## Tech Stack & Architecture
- **Framework:** Next.js 15 (App Router, Server Components, Metadata API).
- **Styling:** Tailwind CSS 4, Framer Motion (animations), Lenis (smooth scroll).
- **Data:** Flat-file CMS (`data/brain.ts`, `data/logs.ts`, `data/projects.ts`).
- **AI Integration:** Gemini 2.5 Flash via AI SDK, utilizing tool-based retrieval (RAG-lite).
- **Analytics:** Lighthouse 99+ Performance Score.

## Key Features
- **meBot (Digital Twin):** Agentic AI assistant with proactive, tool-based retrieval and personality-driven "Architect's Wit."
- **Research Ledger:** A custom-built, markdown-rendered blog system with optimized hero imagery and SEO metadata.
- **UX Principles:** 
    - **3E Framework:** Empathy, Efficiency, and Engagement.
    - **Session Persistence:** AI chat history survives route navigation via `sessionStorage`.
    - **Performance Optimization:** Component-level splitting (Server/Client) for high hydration scores.
    - **Accessibility:** High-fidelity animations with respect for user preferences.

## Roadmap
- **Phase 6:** Research Log Management System (Supabase/NextAuth integration).
- **Phase 7:** Enhanced data visualization for project impact metrics.
